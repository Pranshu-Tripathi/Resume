# Resume

A place storing my different resumes and maybe templates of implementation.

- Find resumes of respective years and versions in the yearly folders created.
- depreciated folder contains dumps and not relevant at the moment resumes.

## Create New Resume

`latex` folder contains a dockerised setup to run latex and create resume out of the provided files.

- use the latex folder containing relevant files for creating resume.
- ./latex-watch.sh <build_dir> [main.tex]
- real time preview can be seen once the file is created as main.pdf in provided <build_dir>
